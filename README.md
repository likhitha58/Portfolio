# 💼 Likhitha's Premium Portfolio

A modern, high-impact, recruiter-attracting portfolio built with **React**, **Tailwind CSS v4**, and **Framer Motion**. This project showcases a blend of full-stack engineering expertise and a strong design sense, specifically tailored for FinTech and AI/ML roles.

---

## 🌟 Key Features

- **Premium Design Aesthetics**: A sleek, dark-themed hero section transitioning into light case-study sections, following 2025 UI/UX trends.
- **Micro-Interactions**: Custom animated cursor and smooth scroll-triggered animations using Framer Motion.
- **Glassmorphism**: Elegant use of backdrop-blur and glass-card components for a state-of-the-art feel.
- **Project Showcase**: Detailed case studies with problem/solution/impact breakdowns, specifically highlighting MERN and AI/ML projects.
- **Responsive Layout**: Optimized for mobile, tablet, and desktop viewing.

---

## 🚀 Tech Stack

- **Frontend**: React (Functional Components & Hooks)
- **Styling**: Tailwind CSS v4 (with custom `@theme` tokens)
- **Animations**: Framer Motion
- **Icons**: React Icons (Fa, Si)
- **Build Tool**: Vite
- **Deployment Ready**: Netlify / Vercel

---

## 🔧 Installation & Setup

1. **Clone the repository**:
   ```bash
   git clone https://github.com/likhitha58/Portfolio.git
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run development server**:
   ```bash
   npm run dev
   ```

4. **Build for production**:
   ```bash
   npm run build
   ```
